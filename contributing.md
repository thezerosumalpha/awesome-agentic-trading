# Contributing to Awesome Agentic Trading

Thanks for your interest in contributing! This list aims to be the definitive resource for the agentic trading space.

Please note that this project is released with a [Contributor Code of Conduct](code-of-conduct.md). By participating in this project you agree to abide by its terms.

## How to Contribute

1. Fork the repo.
2. Add your resource in the appropriate category.
3. Submit a PR with a brief description of why it belongs.

## Format

Use the established list item format:

```
- [Name](https://github.com/user/repo) - Description starting with uppercase and ending with a period.
```

- **One resource per PR** — makes review easier.
- **Include a concise description** — what it does, not marketing copy.
- **Link to the repo or primary source**, not blog posts about it.
- **Check for duplicates** before submitting.

## What Qualifies

- Open-source repos related to AI agents + trading/finance.
- Standards, protocols, and specifications for agentic trading.
- Research papers on LLM/agent-based trading (peer-reviewed or notable preprints).
- Infrastructure tools specifically relevant to agent execution (connectivity, risk, state management).
- Data tools designed for agent consumption (MCP servers, structured APIs).

## What Doesn't Qualify

- General ML/DL trading with no agent component.
- Closed-source products without public documentation.
- Abandoned repos with no commits in 2+ years (unless historically significant).
- Self-promotional links without meaningful open-source contribution.
- Crypto token projects or DeFi yield farming bots.

## Suggesting a New Category

If your resource doesn't fit an existing category, propose a new one in your PR description. The list structure should evolve as the space evolves.

## Questions?

Open an issue. Let's build the map of this space together.
